# Data Analytics Internship Projects

This repository contains my Data Analytics internship projects completed as part of the DecodeLabs Industrial Training Program.

## Projects Included

### Project 1: Data Cleaning & Preparation
Cleaned raw data by handling missing values, removing duplicates, correcting formats, and preparing a submission-ready dataset.

### Project 2: Exploratory Data Analysis
Performed descriptive statistics, trend analysis, outlier detection, and summarized key business observations.

### Project 3: SQL Data Analysis
Used SQL queries including SELECT, WHERE, GROUP BY, ORDER BY, COUNT, SUM, and AVG to extract business insights.

### Project 4: Data Visualization
Created dashboards and charts to communicate insights clearly using storytelling with data.
